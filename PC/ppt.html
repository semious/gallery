Matrix

变形矩阵:matrix(a,b,c,d,e,f)
四种基本变形方式与Matrix之间的关系
 
translate(x, y) = matrix(1, 0, 0, 1, x, y)
scale(x, y) = matrix(x, 0, 0, y, 0, 0)
skewx(a) = matrix(1, 0, tan(a), 1, 0, 0)
skewy(a) = matrix(1, tan(a), 0, 1, 0, 0)
