Transform2D的四种基本变形方式

  • 平移(translate)
  • 伸缩(scale)
  • 歪斜(skew)
  • 旋转(rotate)

Matrix

变形矩阵:matrix(a,b,c,d,e,f)
四种基本变形方式与Matrix之间的关系
 
translate(x, y) = matrix(1, 0, 0, 1, x, y)
scale(x, y) = matrix(x, 0, 0, y, 0, 0)
skewx(a) = matrix(1, 0, tan(a), 1, 0, 0)
skewy(a) = matrix(1, tan(a), 0, 1, 0, 0)
Rotate去哪里了？
rotate是个复合的方法
scale + skew
scale(cos(30deg),cos(30deg)) skew(-30deg,30deg) rotate(30deg)
rotate(a) = matrix(cos(a), sin(a), -sin(a), cos(a), 0, 0)
渲染引擎实际如何处理css变形的？
  1. 首先读取样式
  2. 将所对应的样式转换成matrix变形矩阵
  3. 对于多个样式所对应的矩阵进行相乘，计算出最终的matrix变形矩阵
  4. 将最终的matrix变形矩阵进行相应的图形渲染

已上面的例子为例

  • 将scale(cos(30deg),cos(30deg))矩阵化matrix(cos(30deg),0,0,cos(30deg),0,0)
  • 将skew(-30deg,30deg)矩阵化matrix(1,tan(30deg),tan(-30deg),1,0,0)
  • 将以上两个矩阵相乘，最终得到matrix(cos(30deg),sin(30deg),-sin(30deg),cos(30deg))
  • 将当前的坐标点和以上最终的matrix变形矩阵相乘，最后得到最终位置
[x,y,1]*[matrix(cos(30deg),sin(30deg),-sin(30deg),cos(30deg),0,0,1)]= [x*cos(30deg)-y*sin(30deg)+0,ysin(30deg)+x*cos(30deg)+0,0+0+1]
