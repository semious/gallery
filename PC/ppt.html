渲染引擎实际如何处理css变形的？
  1. 首先读取样式方法
  2. 将所对应的样式转换成matrix变形矩阵
  3. 对于多个样式所对应的矩阵进行逐个相乘，计算出最终的matrix变形矩阵
  4. 将最终的matrix变形矩阵进行相应的图形渲染

已上面的例子为例 - 渲染引擎处理transform的过程

  • 将scale(cos(30deg),cos(30deg))矩阵化matrix(cos(30deg),0,0,cos(30deg),0,0)
  • 将skew(-30deg,30deg)矩阵化matrix(1,tan(30deg),tan(-30deg),1,0,0)
  • 将当前的2d坐标矩阵分别乘以以上两个矩阵，最终得到matrix(cos(30deg),sin(30deg),-sin(30deg),cos(30deg))
  • 将最终的matrix变形矩阵换算成相应的2d坐标，最后得到最终位置
[x,y,1]*[matrix(cos(30deg),sin(30deg),-sin(30deg),cos(30deg),0,0,1)]= [x*cos(30deg)-y*sin(30deg)+0,ysin(30deg)+x*cos(30deg)+0,0+0+1]

Matrix3D和3D变形方法之间的关系
Translate matrix — matrix3d(1,0,0,tX,0,1,0,tY,0,0,1,tZ,0,0,0,1)
Scale matrix — matrix3d(sX,0,0,0,0,sY,0,0,0,0,sZ,0,0,0,0,1)
RotateX(a) matrix — matrix3d(1,0,0,0,0,cos(a),sin(a),0,0,sin(-a),cos(a),0,0,0,0,1)
RotateY(a) matrix — matrix3d(cos(a),0,sin(-a),0,0,1,0,0,sin(a),0,cos(a),0,0,0,0,1)
